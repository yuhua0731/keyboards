## 快捷键

| 按键 | 功能 |
| --- | --- |
| Tab | 轴灯开关 |
| Q | 切换轴灯模式 |
| Caps | 氛围灯开关 |
| A | 切换氛围灯模式 |
| `\` | 输入灯开关 |
| Enter | 单键灯光开关 |
| ↑ / ↓ | 亮度 + / - |
| ← / → | 上一首 / 下一首 |
| E | 2.4G |
| R / T / Y | 蓝牙 1 / 2 / 3 |
| U | USB |
| F | 配对 |
| K | 深度休眠 |
| O | 复位 |
| Space | 播放 / 暂停 |
| Z / X | 音量 + / - |
| C | 静音 |
| WIN | 锁 WIN |
| V / B / N | 层 1 / 2 / 3 |
| M | MAC |
| Back | 计算器 |

## 备注

- `.exe` 为 Windows 配置工具。
- 三个安装包分别是默认版、按键减慢专用版、降低 Alice 氛围灯亮度版。
